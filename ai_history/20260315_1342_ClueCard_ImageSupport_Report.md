## 2. 구현 내용

### 2-1. 단서 데이터 모델 확장

- `src/types/game.ts`
  - `Clue.imageUrl?: string`
  - `ClueCard.imageUrl?: string`
- 단서가 이미지 URL을 저장할 수 있도록 타입 확장

### 2-2. 저장 데이터 정규화 보강

- `src/lib/game-normalizer.ts`
  - 장소 / 단서 / 단서 조건 / 단서 카드 정규화 함수 추가
  - 오래된 저장본과 섞여도 `imageUrl`이 안전하게 정리되도록 보정

### 2-3. 메이커 단서 편집기 개선

- `src/app/maker/[gameId]/edit/_components/LocationEditor.tsx`
  - 단서 생성 기본값에 `imageUrl` 추가
  - 단서 폼에 `단서 이미지 URL` 입력 칸 추가
  - 이미지가 있을 때 즉시 미리보기 표시
  - 접힌 헤더에서 `이미지` 배지로 상태 확인 가능

### 2-4. 플레이어 인벤토리 카드형 표시

- `src/app/play/[gameId]/[charId]/page.tsx`
  - 인벤토리 목록에 이미지 썸네일 추가
  - 상세 모달 상단에 큰 이미지 표시
  - 이미지가 없는 단서는 텍스트 카드로 유지

### 2-5. 문서 반영

- `README.md`
  - 현재 구현 상태에 단서 이미지 지원 반영
