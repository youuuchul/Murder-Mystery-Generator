# Color Mockups

앱 코드에 바로 적용하지 않은 컬러 방향 비교용 HTML 목업들입니다.

- `a-gold-sage-red/index.html`
  - 골드 + 세이지 + 레드 분리안
- `b-teal-product/index.html`
  - 앱스러운 청록 포인트안
- `c-muted-brass/index.html`
  - 더 눌린 브라스/차콜안

각 폴더의 `index.html`은 지금 메이커 편집 화면에서 문제 된 요소를 중심으로 만든 비교용 샘플입니다.
- 상단 단계 네비
- 경고/오류/완료 배지
- 메인/보조 액션 버튼
- 안내 박스 / 토큰 스와치
