### AI 제작도우미 출력 모드 1차 분리

- `chat` 요청에 대해 `guide / draft` 응답 모드 분리
- 입력창에 `자동 / 가이드 / 문안` 모드 토글 추가
- `자동` 모드는 질문 키워드 기준으로 서버에서 `guide / draft` 추론
- `draft` 응답은 `title / body / notes` 구조로 파싱하고, UI에서 붙여넣기용 본문 카드로 별도 렌더
- 빠른 액션은 기존처럼 `guide` 응답으로 유지
- assistant step 번호 범위를 `1~6` 으로 정리해서 엔딩 step도 정상 참조
