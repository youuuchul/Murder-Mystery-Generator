# 2026-03-19 홈 진입 가이드 백로그

## 목적

홈 화면 또는 라이브러리 첫 진입 시,
사용자가 “무엇을 어디서 시작해야 하는지” 바로 이해할 수 있도록
기본 사용 가이드를 제공하는 백로그다.

현재는 기능은 많아졌지만,
처음 들어온 사용자가 `게임 만들기`, `게임 플레이`, `GM 진행` 중 무엇을 눌러야 하는지 바로 파악하기 어렵다.

## 필요한 방향

### 1. 상단 탭 또는 진입 분기

최소한 아래 두 갈래는 분리돼야 한다.

- 게임 만들기
- 게임 플레이

확장안:

- 게임 만들기
- GM 진행
- 플레이 참여

## 추천 UX

### 홈 / 라이브러리 상단 가이드 탭

- `게임 만들기`
- `게임 플레이`

탭을 누르면 각자 다른 짧은 가이드 문서 또는 안내 패널을 보여준다.

### 게임 만들기 가이드에 들어갈 내용

- 라이브러리에서 새 게임 만드는 법
- Step 1~6 구조 설명
- 저장 방식
- 이미지 업로드
- AI 제작도우미 사용법

### 게임 플레이 가이드에 들어갈 내용

- `/join` 접속
- 코드 입력
- 캐릭터 선택
- 장소 탐색 / 인벤토리 / 투표 기본 흐름
- 재접속 방법

## 구현 후보

### 1안. 라이브러리 상단 도움말 탭

- 가장 간단함
- 현재 구조를 크게 바꾸지 않아도 된다

### 2안. 별도 `/guide` 또는 `/help` 페이지

- 문서량이 많아지면 확장에 유리
- 홈 화면은 요약, 상세는 전용 페이지로 분리 가능

### 3안. 첫 방문 온보딩 패널

- UX는 좋지만 구현 범위가 더 크다
- 현재 단계에서는 우선순위 낮음

## 권장 순서

1. 라이브러리 상단에 `사용 가이드` 진입점 추가
2. `게임 만들기 / 게임 플레이` 2채널 분리
3. 필요하면 이후 `GM 진행` 채널 추가

## 우선순위

- 중간
- 실제 유저 테스트 전에 있으면 좋고, 테스트 피드백 수집에도 도움이 된다.
